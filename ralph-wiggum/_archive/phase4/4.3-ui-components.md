# Spec: Phase 4.3 - UI Components

> **Goal**: Create the UI components for managing boards (Create, Rename, Delete).
> **Context**: We need reusable components for board operations that can be used in the sidebar.

## 1. Requirements

### 1.1 Create Board Dialog
- **File**: `OpenKanban/src/features/boards/components/create-board-dialog.tsx`
- **Props**: `parentId: string`, `children: React.ReactNode` (trigger).
- **Behavior**:
    - Renders a `Dialog` with a form.
    - Field: "Board Name" (required).
    - On Submit: Calls `useCreateBoard`.
    - Payload: `{ name: "...", filters: { parentId }, columnConfig: [...] }`
    - Default Column Config: Backlog, In Progress, Done.

### 1.2 Board Actions Menu
- **File**: `OpenKanban/src/features/boards/components/board-actions-menu.tsx`
- **Props**: `boardId: string`, `boardName: string`.
- **Behavior**:
    - Renders a `DropdownMenu` (trigger: 3-dots icon or passed as child).
    - Items: "Rename", "Delete".
    - "Rename" opens `RenameBoardDialog`.
    - "Delete" opens `DeleteBoardDialog`.

### 1.3 Delete Board Dialog
- **File**: `OpenKanban/src/features/boards/components/delete-board-dialog.tsx`
- **Props**: 
    - `boardId: string`
    - `projectId: string` (Required for redirect logic)
    - `open: boolean`
    - `onOpenChange: (open: boolean) => void`
- **Behavior**:
    - Warning message: "This action cannot be undone."
    - On Confirm: Calls `useDeleteBoard`.
    - **Logic**: Check if `pathname` contains the deleted `boardId`. If so, redirect to `/project/${projectId}`.

### 1.4 Rename Board Dialog
- **File**: `OpenKanban/src/features/boards/components/rename-board-dialog.tsx`
- **Props**: `boardId: string`, `currentName: string`, `open: boolean`, `onOpenChange`.
- **Behavior**:
    - Input with current name.
    - Calls `useUpdateBoard`.

## 2. Definition of Done
- [ ] Components render without errors.
- [ ] Dialogs open/close correctly.
- [ ] Mutations are called with correct parameters.
- [ ] Redirect happens after deleting active board.

## 3. Implementation Plan
1.  Create `create-board-dialog.tsx` (Use `CreateProjectDialog` as reference).
2.  Create `delete-board-dialog.tsx`.
3.  Create `rename-board-dialog.tsx`.
4.  Create `board-actions-menu.tsx` to orchestrate them.
